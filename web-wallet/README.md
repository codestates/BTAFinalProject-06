## Getting Started

패키지 설치
```bash
npm install
# or
yarn install
```

프로그램 실행
실행시 localhost:3030으로 실행됨
```bash
npm run dev
# or
yarn dev
```

.env.sample을 .env로 변경<br/>
아래 값은 hedera portal에 가입하여 추가 필요.
https://portal.hedera.com/
```
NEXT_PUBLIC_WALLET_ACCOUNT_ID=
NEXT_PUBLIC_WALLET_PRIVATE_KEY=
```
